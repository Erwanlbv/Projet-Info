Installation

Cet outil utilise plusieurs bibliothèques: DJL et JavaFX

Si vous utilisez Eclipse ou IntelliJ, il faut de plus faire les étapes suivantes: 

Pour lancer le projet, il faut exécuter la classe Main avec une configurations particulière. 
Dans Run Configurations, il faut ajouter une configuration avec le path du fichier Main comme Base Directory et il faut ajouter comme Goals/exécution: compile exec:java -f pom.xml
